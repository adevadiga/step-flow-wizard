# step-flow-wizard

An intelligent multistep wizard for React

### Install


```
npm install react-step-wizard

yarn add react-step-wizard
```

### Import Component

```js
import StepFlowWizard from "step-flow-wizard";
```

### JSX Syntax

Pass a config array containing components.

```jsx
<StepFlowWizard 
  screenConfig={step}/>
```
